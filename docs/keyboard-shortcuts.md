# キーボードショートカット

| キー | 動作 |
|---|---|
| `Ctrl+Shift+O` | ルートフォルダを追加 (フォルダ選択 + JSON 断片を案内) |
| `Ctrl+R` | 設定ファイルを再読み込み |
| `Ctrl+B` | エクスプローラ (左ペイン) の表示/非表示切替 |
| `Alt+Z` | 折り返し (Word Wrap) のオン/オフ |
| `Ctrl+W` | 現在のタブを閉じる |
| `Enter` | ツリーで選択中のファイル/フォルダを開く/展開 |
| `(マウス) ダブルクリック` | 同上 |
| `(マウス) ミドルクリック (タブ)` | そのタブを閉じる |
| `(マウス) 左クリック (タブの×)` | そのタブを閉じる |
| `(マウス) 右クリック (ツリー)` | コンテキストメニュー表示 |

## コンテキストメニュー (ツリー右クリック)

- 開く
- 別なアプリで開く ▶ (`externalEditors` から動的に構築)
- テキストとして強制で開く (バイナリ判定をスキップ)
- エクスプローラで表示 (`/select` で対象を選択した状態)
- パスをコピー
- 再読み込み (フォルダ配下を再列挙)
