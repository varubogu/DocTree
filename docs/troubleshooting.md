# Troubleshooting

## 設定ファイルが読み込まれない

- 起動時に「設定ファイルの読み込みに失敗しました」と表示される場合は
  JSONC のパースエラーです。エラーメッセージ中の行/桁を頼りに `settings.jsonc` を修正してください。
- 修正後、メニュー [ファイル] > [設定を再読み込み] (Ctrl+R) で反映できます。
- 再生成したい場合は `%AppData%\DocTree\settings.jsonc` を削除して再起動すると、
  埋め込みリソースから初期テンプレが書き出されます。

## ファイルが開けない

| メッセージ | 原因 / 対処 |
|---|---|
| 「テキスト拡張子のホワイトリストに含まれていません」 | `settings.jsonc` の `textExtensions` に対象拡張子を追加。または右クリック > [テキストとして強制で開く] |
| 「バイナリと判定されたため開きません」 | ヌルバイトを検出。本当にテキストなら強制で開いてください |
| 「アクセスが拒否されました」 | 権限不足。権限を確認、または別アカウントで実行 |
| 「ファイルサイズが上限を超えています」 | 既定 64 MiB。本当に開く必要があるなら別エディタを使用 |

## ネットワークドライブ (UNC / マップドライブ) の動作が遅い

- 初回フォルダ展開時にネットワーク I/O が走るため、リモートでは一拍待ちが入ります。
- DocTree はリモートと判定したフォルダのみ **バックグラウンドで列挙** します
  (UI はブロックされません)。
- それでも遅い場合は、回線そのものまたは SMB 設定 (キャッシュ等) を確認してください。
- 接続できないネットワークパスは、ツリー上に「(パスが見つかりません)」と表示されます。

## OneDrive / クラウド同期フォルダで固まる

クラウドオンリー (オフライン化された) ファイルを開くとダウンロードが発生します。
固まる場合は、いったんエクスプローラから対象を「常にこのデバイスに保持」してから開いてください。

将来のバージョンで設定からスキップ可能にする予定です。

## 文字化けする (日本語ファイル)

DocTree は **UTF-8 / UTF-16 のみ** 対応です。Shift_JIS は対応しません。
詳細は [encoding.md](encoding.md) を参照。

対処:
1. ソースを UTF-8 (BOM付きでも可) に変換する (推奨)
2. もし「全部 Shift_JIS でしか持てない」状況なら、
   外部エディタで開く設定 ([external-editors.md](external-editors.md)) を活用してください

## シンボリックリンク・ショートカットが見えない

仕様です。詳細は [symlinks-and-shortcuts.md](symlinks-and-shortcuts.md) を参照。
リンク先のフォルダを `roots` に直接追加して回避してください。

## アプリ状態 (タブ復元等) を初期化したい

`%AppData%\DocTree\state.json` を削除して再起動してください。
設定 (`settings.jsonc`) には影響しません。
